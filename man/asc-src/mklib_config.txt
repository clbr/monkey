mklib_config(3)
=============

NAME
----
mklib_config - configure your libmonkey context

SYNOPSIS
--------
*#include <monkey.h>*

*int mklib_config(mklib_ctx, ...);*

DESCRIPTION
-----------
Yay

RETURN VALUE
------------
On success, returns a valid mklib_ctx. On failure NULL is returned.

SEE ALSO
--------
*mklib_init*(3), *mklib_vhost_config*(3), *mklib_start*(3), *mklib_stop*(3)

RESOURCES
---------
